## ノード座標行列
ノード1のx座標, ノード1のy座標  
ノード2のx座標, ノード2のy座標  
...  
...  
ノードNのx座標, ノードNのy座標  

例)  
c = [[0, 4],  
    [2, 4],  
    [1, 2],  
    [3, 1],  
    [4, 3]]  

## リンク行列
(i, j)成分にノードi→ノードjの重み  
(i, i)成分=同じ地点同士は0  
一方通行や道路なしなど通行不可の場合も0で良い(infにする必要はない)  
  
例)  
l = [[0, 6, 3, 0, 0],  
    [0, 0, 0, 1, 2],  
    [2, 1, 0, 3, 0],  
    [0, 0, 2, 0, 1],  
    [0, 3, 0, 0, 0]]
